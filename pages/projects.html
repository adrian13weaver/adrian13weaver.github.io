Command Line BattleShips

I made this command line BattleShips game in C# with .NET. It was made about a year ago as of writing this, so please ignore any issues.

Inside this repo is a Executables folder with both the Mac OS and Windows 64 executables in a zip format. Simply unzip and run.

This allows either 2 player to play against one another on one computer, or a single player can choose to play against the computer.
